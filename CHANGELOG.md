## [3.1.0] - 2026-07-22

### Added

- NVIDIA NIM integration with support for Meta Llama 3 and DeepSeek model variants
- Groq provider support for low-latency LPU inference
- OpenRouter provider support for aggregated model routing
- Memory Manager with persistent AI persona and knowledge base
- React 19 dark glassmorphism control panel UI
- DOM-based chat message listener for Free4Talk text chat rooms
- Multi-provider dynamic failover and auto-switching
- Configurable welcome message for new room participants
- Extension icons for all supported AI providers
- Manifest V3 permissions scoped to free4talk.com

### Changed

- Optimised DOM query selectors for reduced latency during chat parsing
- Enhanced memory management interface with filterable item states and deletion controls
- Refined exception handling for HTTP 429 and 503 response codes
